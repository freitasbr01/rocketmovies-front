## 💻 Sobre o projeto

A plataforma permite aos usuários criar uma conta pessoal, através da qual podem acessar e personalizar sua própria lista de filmes favoritos. Além disso, oferece a funcionalidade única de avaliar cada filme que o usuário assistiu, proporcionando uma experiência de usuário rica e personalizada. Esta aplicação é perfeita para qualquer pessoa que queira manter um registro de todos os filmes que assistiu e compartilhar suas opiniões sobre eles.”

Projeto desenvolvido durante o curso Explorer da **Rocketseat**.
